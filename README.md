# matratzen24-lintfort.de

Astro website for Matratzen Outlet Kamp-Lintfort.

## Stack

- Astro 5
- Tailwind CSS 4
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run check
```

## Notes

Main site content lives in `src/lib/site.ts`.
